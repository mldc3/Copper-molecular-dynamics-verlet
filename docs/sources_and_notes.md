# Sources and Notes

This repository is based on computational physics coursework and class notes.

Primary topics covered in the implementation and analysis are:
- FCC crystal generation
- Lennard-Jones potentials
- Periodic boundary conditions
- Verlet and velocity-Verlet methods
- Thermodynamic observables
- OVITO visualization

The current repository structure is intended as a clean portfolio presentation of that coursework, preserving the original numerical setup while improving organization and documentation quality.
