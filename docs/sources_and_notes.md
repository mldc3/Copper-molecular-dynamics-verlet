# Sources and Notes

This repository is based on computational physics coursework and accompanying study notes. The implementation and documentation focus on standard methods used in introductory and intermediate molecular dynamics practice.

## Core topics covered

- numerical integration of ordinary differential equations,
- Verlet and velocity-Verlet methods,
- crystalline lattices,
- FCC structure,
- Lennard-Jones potentials,
- periodic boundary conditions,
- minimum image convention,
- molecular dynamics observables,
- radial distribution functions,
- OVITO visualization.

## Notes on scope

The repository is organized to support transparent, reproducible scientific programming. It emphasizes method clarity and diagnostic interpretation rather than claiming experimental agreement or publishing-scale material parameter fitting.

## Suggested references for implementation context

- Standard computational physics lecture notes on MD and symplectic integration.
- Introductory atomistic simulation texts covering pair potentials and periodic simulation cells.
- OVITO documentation for trajectory visualization workflows.
