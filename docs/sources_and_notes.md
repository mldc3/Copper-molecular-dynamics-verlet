# Sources and Notes

This project is based on computational physics coursework and class notes focused on atomistic simulation methods.

Topics used in this repository include:
- numerical integration of Newtonian dynamics,
- Verlet and velocity-Verlet methods,
- FCC crystal generation,
- Lennard-Jones pair potentials,
- periodic boundary conditions,
- minimum image convention,
- molecular dynamics observables (energy, temperature),
- radial distribution functions and OVITO-based visualization.

Important modeling note: the Lennard-Jones potential is used here as an educational simplified model for copper. It is useful for learning MD workflows, but it is not a production-quality metallic potential for quantitative copper materials prediction.
